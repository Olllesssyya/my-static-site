Готовий деплой

Зміни у GitHub → автоматичне оновлення сайту за 1–2 хвилини.

Глобальна доставка

Контент розподіляється через CDN для швидкого доступу по всьому світу.

Простота

Жодних фреймворків: лише index.html, style.css, script.js.

Про проєкт

Цей шаблон створено спеціально для Azure Static Web Apps без збірки. Можна безпечно додавати сторінки, зображення та стилі — усе розгортається автоматично.

  • Маршрути працюють «як є» (наприклад, /about.html).
  • Додатково можна підключити API на Azure Functions.
  • Кастомні домени, SSL, захист та ін. — у кілька кліків.

Швидкий старт редагування

  1. Відкрий index.html та зміни тексти/розділи.
  2. Онови стилі у style.css.
  3. Зроби commit → деплой автоматично запуститься.
